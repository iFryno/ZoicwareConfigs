## Windows 11 Safe Config (safe for most users)

- All registry tweaks included in Zoicware [Read More](https://github.com/zoicware/ZOICWARE/blob/main/registrytweaks.md)
- Install essential packages such as C++, DirectX and .NET 3.5
- Disable telemetry
- Full Windows debloat leaving only Paint, Photos, Notepad, Terminal, Store and XBOX
- Black Windows theme
- Disable security warning when opening downloaded files
- Set mouse and sound schemes to none
- Remove network icon from File Explorer
- Disable PowerShell logging
- Enable faster shutdown and restart
- Use More accurate time server for system clock
- Add Take ownership to context menu
- Remove Add to Favorites, Customize this folder, Give access to, Open in Terminal, Restore previous versions, Print, Send to, Share, Troubleshoot compatibility and Include in library from context menu
- Set most services to manual
- Show all taskbar tray icons
- Remove Quick Settings tiles
- Disable Notepad tabs and rewrite
- Remove ads from Settings app

## Windows 11 Extreme Config (not safe for most users)
**Same as above plus added features below**

- Block Razer and ASUS download servers
- Enable No GUI Boot
- Disable BitLocker device encryption
- Enable Windows 10 File Explorer ribbon
- Set background mouse throttle to 50 Hz
- Enable 24H2 Start menu


## How to Use

- **1.** Download the latest version of Zoicware under [releases](https://github.com/zoicware/ZOICWARE/releases)
- **2.** Download your desired config
- **3.** Run Zoicware and navigate to `Utilities` > `Import/Export Config`
- **4.** Choose the location of the config file and click `Run Tweaks`
